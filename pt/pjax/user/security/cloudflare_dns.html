O que é DNS

O Sistema de Nomes de Domínio (Domain Name System – DNS) permite-lhe acessar o site através de um endereço Web amigável, tal como www.binary.com, em vez de um endereço IP complicado.

Por que deve usar o 1.1.1.1 em vez do serviço DNS padrão do seu provedor de Internet

Os serviços DNS padrão disponibilizados pelo seu provedor de Internet poderão ser lentas e inseguras. A Binary.com recomenda a utilização do serviço DNS da Cloudflare, que lhe proporciona um método mais rápido e privado de navegar na Internet.
