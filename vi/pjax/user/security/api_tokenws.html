Mã API Token

Để truy cập vào các ứng dụng di động của chúng tôi và các ứng dụng của bên thứ ba, bạn cần phải tạo ra một mã token API.

  • Chỉ cần nhấp vào nút "Tạo" để tạo ra các mã token của bạn; sau đó sao chép và dán nó vào ứng dụng.
  • Chọn kiểu token API bạn cần, dựa trên khả năng mà bạn muốn làm cho nó khả dụng.
