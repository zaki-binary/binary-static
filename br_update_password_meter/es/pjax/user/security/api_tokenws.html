Token API

Para acceder a nuestras aplicaciones móviles y otras aplicaciones de terceros, primero debe generar un token de la API.

  • Simplemente haga clic en "Crear" para generar el token; a continuación, cópielo y péguelo en la aplicación.
  • Elija el tipo específico de token API que necesita, teniendo en cuenta las características que desea habilitar.
