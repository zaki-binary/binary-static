Что такое DNS

Система Domain Name System (DNS) открывает доступ на вебсайт через простой веб адрес, например, www.binary.com, вместо сложного IP адреса.

Преимущества 1.1.1.1 по сравнению с сервисом DNS по умолчанию от вашего интернет-провайдера

Сервисы DNS по умолчанию, предоставленные вашим интернет-провайдером, могут оказаться медленными и ненадёжными. Binary.com рекомендует пользоваться сервисом DNS от Cloudflare 1.1.1.1, предоставляющим более быстрое и надёжное соединение с интернетом.
