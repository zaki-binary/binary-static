Projets open source Binary.com

Binary.com soutien activement le mouvement open source et autorise actuellement l'accès à certains éléments de son code base dans le cadre de projets open-source.

Contenu statique

Améliorez le contenu frontal de Binary.com

Le contenu statique du site de Binary.com (CSS, Javascript) est disponible dans le projet open source binary-static sur github. Vous voulez modifier l'apparence du site Binary.com, ou améliorer la comptabilité du navigateur, la vitesse d'affichage ou la performance Javascript ? Participez au projet en envoyant des demandes de tirage (pull requests) pour les modifications de code que vous souhaitez suggérer.

Modules Perl

Contributions de Binary.com sur le CPAN

Binary.com est un membre actif de la communauté Perl et archive actuellement un certain nombre de ses modules internes sur le site du CPAN. Si vous êtes passionné de Perl, vous pouvez apporter votre contribution en révisant ces modules et en envoyant des demandes de tirage (pull requests) via github.
